启用Win10的Linux子系统

冰是睡着的水
2018-08-30
Linux相关

对于使用windows的用户来说，学习Linux最方便的方法就是安装虚拟机，然后在虚拟机里安装Linux，毕竟要在同一台电脑上安装windows和Linux还是有一定难度的，但即使是在虚拟机里安装Linux也有一堆的参数需要设置，对于不太了解的人来说，还是比较繁琐的。不过在Win10系统里内置了一个Linux的子系统，只需要通过几步简单的设置就可以开启，这里就来记录一下我开启的过程。

打开设置，找到 更新和安全，找到 开发者选项，打开 开发人员模式，

打开控制面板，找到 卸载程序，

点击 启用或关闭 Windows 功能，勾选 适用于 Linux 的 Windows 子系统，选择完成后 确定，

等待一段时间，然后 重启，

重启后打开命令行，输入 lxrun /install /y 安装Linux，

等待一段时间，

安装完成后，在命令行输入 bash 进入Ubuntu，进入以后就可以使用了，默认root用户，没有密码，如果需要的话也可以设置，

以上就是在Win10启用Linux的方法，后续可能会有补充。

2019年2月12日更新

如果输入 lxrun /install /y 不能安装的话可以打开应用商店，搜索 wsl 会搜索到几个可选的Linux系统，选择一个，

然后和平时安装应用一样的安装，

安装完成后在开始菜单可以找到，

有其他的以后在更新。
